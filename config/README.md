## config.json 文件

`emails`： 邮件列表，这里配置接收邮件的pop3服务器和账号密码

`emails.pop3`: 接收邮件的POP3服务器地址

`emails.port`：POP3的端口号

`emails.enable_ssl`: 是否使用 ssl 

`emails.accout`: 邮箱账号

`emails.passwd`: 邮箱密码

*************************

`white_list`: 白名单列表, 只处理白名单中发过来的消息

***************************************

`db`: 数据库的配置

`db.path`: 数据库的路径

`db.name`: 数据库名字

*********************************
`publish_path`: 文章存放的路径
